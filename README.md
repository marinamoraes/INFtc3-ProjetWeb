# INFtc3 ProjetWeb
 
Groupe de TD A2b, Groupe B (Asie)

1) Copiez le dossier "Site Final" à l'endroit de votre choix dans votre ordinateur

2) Exécutez le fichier Projet_serveur.py (à l'aide de spyder ou d'un autre terminal)

3) Ouvrez le navigateur de votre choix à l'adresse suivante : "http://localhost:8080/TourdAsie.html"

L'application web affiche des pointeurs sur chaque pays d'Asie. Il est possible en appuyant sur un des pointeurs de faire apparaître à gauche de l'écran les informations du pays concerné, et en tapant le nom d'un pays dans la barre de recherche, d'obtenir aussi ses informations.
