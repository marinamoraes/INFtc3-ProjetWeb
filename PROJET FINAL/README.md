Instructions pour démarrer l'application :

1) Copiez le dossier "Projet Final" à l'endroit de votre choix dans votre ordinateur

2) Exécutez le fichier Projet_serveur.py (à l'aide de spyder ou d'un autre terminal)

3) Ouvrez le navigateur de votre choix à l'adresse suivante : "http://localhost:8080/TourdAsie.html"




Fonctionnement de l'application : 

L'application web affiche des pointeurs sur chaque capitale des pays d'Asie. Il est possible en appuyant sur un des pointeurs de faire apparaître à gauche de l'écran les informations du pays concerné, et en tapant le nom d'un pays dans la barre de recherche, d'obtenir aussi ses informations. Si vous utilisez la barre de recherche, veillez à écrire le nom anglais du pays.



Pour recréer la base de données : 

1) Vérifiez que le fichier pays.sqlite n'existe pas

2) Exécutez le fichier Bdd2.py (à l'aide de spyder ou d'un autre terminal)
